# React + Vite

## 🚀 Khởi tạo dự án

Cài đặt `pnpm` toàn cục (nếu chưa có):

```bash
npm install -g pnpm
```

Cài đặt các phụ thuộc:

```bash
pnpm install
```

## 🛠️ Khởi động server phát triển

```bash
pnpm run dev
```
